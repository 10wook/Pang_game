# Pang_game
나도코딩 따라하기 팡게임 만들기
해당 게임에 사용된 이미지들은 제가 직접 찍은 사진들과 
Ridho Imam Prayogi 의 운석 그림
Prettycons Flat 의 로켓 그림을 사용하였습니다.

게임을 하시려면
